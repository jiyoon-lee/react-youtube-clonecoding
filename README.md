# Youtube Clone Coding
React 역량강화를 위해

## 주요 기능
- 비디오 리스트 출력
- 비디오 키워드 검색
- 비디오 상세 페이지

## 사용기술
React, TailwindCSS, timeago, axios, react-query, react-router
